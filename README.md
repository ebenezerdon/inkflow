# InkFlow - Handwriting Practice Generator

InkFlow is a modern web application for creating custom handwriting practice sheets. Built by [Teda.dev](https://teda.dev), the AI app builder for everyday problems.

## Features
- **Customizable Guides**: Adjust line height, guide colors, and styles.
- **Multiple Fonts**: Support for dotted (tracing), cursive, and print fonts.
- **AI Integration**: Generate practice sentences using a local LLM (WebLLM) directly in the browser.
- **Print Perfect**: CSS-optimized for perfect A4 printing.
- **Live Preview**: Real-time preview of your worksheet.

## Tech Stack
- HTML5 / CSS3
- Tailwind CSS
- jQuery
- WebLLM (Local AI)

## Usage
1. Open `index.html` to view the landing page.
2. Click "Start Creating" to enter the app.
3. Customize your text and settings.
4. Click "Print / PDF" to save your worksheet.
